simulador de investimentos imobiliários.
Planilha elaborada aplicando conceitos de Excel no desenvolvimento de uma ferramenta prática de simulador de investimentos em fundos imobiliários. O desafio foi construir uma planilha que ajude os usuários a tomar decisões mais informadas sobre seus investimentos. A solução proposta visa automatizar calcúlos complexos, proporcionando ao usuário uma visão clara de seu potencial retorno.
